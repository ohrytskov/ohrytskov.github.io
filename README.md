# ohrytskov.github.io

A static Next.js profile site for `https://github.com/ohrytskov`.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- GitHub Pages via GitHub Actions

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The site uses only public GitHub profile data and intentionally does not render repository lists.
